# Web模块
## web服务器
web服务器的基本功能就是提供web信息浏览服务，他只需要支持HTTP协议，HTML文档格式和URL，与客户端的网络浏览器配合。
大多数web服务器都支持服务端的脚本语言（php, python, ruby）等， 并通过脚本语言从数据库获取数据， 将结果返回给客户端浏览器
目前流行的web服务器是：apache, nginx, iis.
